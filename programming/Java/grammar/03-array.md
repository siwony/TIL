### 배열 사용
**array.length**
> 배열 크기 반환

**for each**  
``for(타입 값을 받아줄 변수명 : 출력하고 싶은 자료구조)``
```java
int[] arr = {1, 2, 3}
for(int i : arr)
    System.out.println(i)
//출력: 
//1 2 3
```
