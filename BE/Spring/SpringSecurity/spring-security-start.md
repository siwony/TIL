# Spring security 시작하자

### 의존성 추가
#### Maven
`pom.xml`
```xml
<dependencies>
  <!-- ... other dependency elements ... -->
  <dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-security</artifactId>
  </dependency>
</dependencies>

```

#### Gradle
`builde.gradle`
```gradle
dependencies {
  compile "org.springframework.boot:spring-boot-starter-security"
}
```
